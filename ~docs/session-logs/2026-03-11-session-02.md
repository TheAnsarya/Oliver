- **HTTP connection pooling** via `SocketsHttpHandler` with `MaxConnectionsPerServer=10`
- **Batch processing** — torrent/image downloads processed in chunks of 100 with DB saves per batch
- **Eliminated all build warnings** — 0 warnings on final build

## Technical Decisions

- **C# 14 `preview`**: Required since .NET 10 is still in preview/RC; once stable, change to `14`
- **SemaphoreSlim over Parallel.ForEachAsync**: More control over batch processing and DB save points
- **Separate scopes per download**: Each parallel download creates its own `IServiceScope` to avoid DbContext threading issues
- **No Polly dependency**: Built retry logic manually — simpler for current needs, avoids dependency
- **Vite 7 over Next.js/Remix**: Simple SPA dashboard, no SSR needed
