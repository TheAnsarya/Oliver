## Architecture

The app runs as a .NET 9 Worker Service with a single background worker (`YtsSyncWorker`) that:

1. Paginates through all YTS movies via the API
2. Upserts movie metadata, genres, and torrent info into SQLite
3. Downloads all torrent files to `YtsData/torrents/`
4. Downloads all cover/background images to `YtsData/images/{id}/`
5. Tracks progress via `SyncState` table for resume support
